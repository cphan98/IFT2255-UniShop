Analyse

UniShop informatisera les fonctions internes du système telles que passer des commandes et lister des produits via Java et OOP, représentant des entités comme Client et Produit. Les données seront sauvegardées sur un disque dur externe, ou un serveur Cloud dans l'idéal. Les utilisateurs nécessiteront un PC avec Java, un disque dur interne, et utiliseront une interface en ligne de commande pour interagir avec UniShop, qui requiert une connexion avec un compte pour accéder pleinement aux fonctionnalités. Dans l'idéal, UniShop interagira avec le serveur pour traiter les données, et peut-être avec les API des revendeurs.
